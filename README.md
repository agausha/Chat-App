# Chat-app

## Install packages

```bash
npm install
#or
yarn install
#or
pnpm install
```

### Setup .env file

```js
DATABASE_URL=
NEXTAUTH_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

GITHUB_ID=
GITHUB_SECRET=

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=

NEXT_PUBLIC_PUSHER_APP_KEY=
PUSHER_APP_ID=
PUSHER_SECRET=

```

### Setup Prisma

````bash
npx prisma db push


### Start the app

```shell
npm run dev
#or
yarn dev
#or
pnpm dev
````
